This a an example taken from 5.2.4 dragon book to evaluate the binary numbers
such as 11.01 = 3.25.
.jpg file is also included as the problem statement.

Just run python binary-parser.py command.
It will ask you for the binary number such as 11.01.
Result will be the equ. decimal answer.

graph.dot file is also generated.
To generate the parse tree corresponding to it. Run:
dot -Tps graph.dot -o graph.ps
